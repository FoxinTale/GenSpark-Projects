Welcome to a project management website. This is designed to help you manage projects from start to finish.
As a user, you are associated with a specific client and are able to manage multiple projects.

Once logged in, you are greeted with a dashboard which provides an overview of all projects, statuses of your projects and any pending tasks.
You can also create new projects and assign team members to a task. Deadlines can also be set to help keep tasks on track.

Each project is divided into separate tasks, of which can be in various stages of completion.

This is a solution anyone who needs to manage multiple projects and tasks for a client.
Being easy to use, intuitive and efficient, it provides all the features needed to stay on top of everything.
